### Build:

1. Frontend:
  - create `.env` next to `.env.example`
  - copy `.env.exaple` to `.env`
2. Backend:
  - create `.env` next to `.env.example`
  - copy `.env.exaple` to `.env`
  - set `IPSTACK_API_KEY`
3. Docker:
 - run `docker compose up --build`
